# datax.cash - LLM-friendly Documentation

## Overview
datax.cash is a free desktop application for deploying and managing data tools across multiple hosts using Docker containers. It provides a Docker app store, multi-host management, and SSH connection pooling.

## Product Information
- **Name**: datax.cash
- **Tagline**: Manage Ur Data, Price Ur Data, Cash Ur Data
- **Type**: Desktop Application
- **Platforms**: Linux (available now), Windows & macOS (coming soon)
- **License**: Free, no registration required
- **Download**: https://github.com/dataxcash/data.x.cash/releases/download/firstrelease/dataX.0.99.1022.linux

## Key Features
1. **Docker App Store**: One-click deployment of pre-configured data tools
   - Databases: PostgreSQL, MySQL, Oracle, IBM DB2
   - Management Tools: pgAdmin, MySQL Client
   - DATAX Tools: X0 Diagnosis, X1 Simulation, M3 Migration, T3 Sync, V3 Comparison
   - License Tools: License Control Server

2. **Multi-Host Management**: Manage multiple remote servers via SSH from a single desktop interface

3. **Data Tools Suite**: Access to professional data migration, synchronization, and comparison tools

4. **Security**: SSH connection pooling with secure access control

5. **Cross-Platform**: Desktop application for Windows, macOS, and Linux
